# DBFoodsRes

#### 介绍
美食网站（重点为登录界面）

#### 软件架构
软件架构说明
1.  MVC设计模式，适合初学者。
2.  以excel作为临时数据库，不涉及第三方数据库操作。
3.  仅作为学习讨论，谢谢！


#### 安装教程

1.  需要通过MyEclipse（推荐MyEclipse for Spring 10/MyEclipse 8.X）在本地开放端口。
2.  本地盘符内需要有相应Excel文件（注册登录功能）
